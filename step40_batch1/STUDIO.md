# Half Space Studio

Studio is the owner-facing home screen for admin mode. It does not create a
second content system: it indexes the existing draft and published data, then
opens the established editor, ranking, XI, media, publishing, or diagnostic
tool for the selected item.

The green admin bar prioritizes Studio, Search, Preview, Undo/Redo, and Publish.
The **More** menu groups lower-frequency controls into Workflow, Site setup,
URLs and discovery, Site health, and Safety and access.

Studio sections are Overview, Content, Rankings, XIs, Media, Publishing, and
Site Health. The public site is unaffected and Studio opens only during a
verified admin session.
