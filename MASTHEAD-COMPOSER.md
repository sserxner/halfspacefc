# Masthead Composer

Open **Admin → Studio → Design → Masthead Composer**.

The homepage starts with a clean green-and-gold masthead. The old flattened collage is not used.

## Build the masthead

- Select any approved figure in the left library to add it as an independent layer.
- Drag a figure to move it. Use its handles or the inspector to resize, rotate, flip, crop, zoom, and change its focal point.
- Use the edge-dissolve, glow, opacity, blend, finish, and layer-order controls to integrate it with the canvas.
- Choose **Edit masthead text** to change the title or tagline wording, size, position, spacing, color, and opacity.
- Switch between **Desktop** and **Mobile** to create separate layouts.
- Use **Add images**, **Paste**, or **Media** to add further images to the reusable library.

## Save safely

- **Save draft** keeps the editable composition private.
- **Save & use on site** creates the finished masthead image as a pending site change.
- Use the normal **Publish Changes** review when it is ready to go live.
- **Reset canvas** removes placed layers and returns to the blank green-and-gold masthead.

Every approved figure remains a separate source asset. Editing the layout never regenerates or permanently changes those library files.
